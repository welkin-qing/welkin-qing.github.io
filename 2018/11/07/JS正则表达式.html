    • 任意字符

    5. 量词

    • { n, } ——- 至少n次
    • （*） ——- （星号） 表示任意次，即{ 0, } 0到任意次，不建议使用
    • ？ ——- 零次或一次 {0,1}
    • {+} —— （加号）表示一次或者任意次{1，}，即最少出现一次
    • {n}——- 正好n次
